Triggers Info Legenda dos triggers

Each trigger subject.txt file is a matrix of 180 event rows x 2 columns
- The first column refers to the event label
- The second column refers to the event onset frame (sampling rate = 3.9 Hz, i.e. approximately 4 frames/second)

Legends:
Event frame values mark the onset of each event. The possible events/labels are:

- 10.2 Biological Motion Stimulus 0.2 fps onset
- 10.4 Biological Motion Stimulus 0.4 fps onset
- 16.4 Biological Motion Stimulus 6.4 fps onset
- 112.8 Biological Motion Stimulus 12.8 fps onset
- 125.6 Biological Motion Stimulus 25.6 fps onset
- 20.2 Non-biological Motion Stimulus 0.2 fps onset
- 20.4 Non-biological Motion Stimulus 0.4 fps onset
- 26.4 Non-biological Motion Stimulus 6.4 fps onset
- 212.8 Non-biological Motion Stimulus 12.8 fps onset
- 225.6 Non-biological Motion Stimulus 25.6 fps onset
- 12 Temporal reproduction onset
- 13 Inter-stimuli interval onset

Each stimulus lasts 15 second (58.5 frames).
